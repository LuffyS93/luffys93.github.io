同时，我们引入一下相对路径与绝对路径的概念。

相对路径与绝对路径

举个例子，你曾经住在某高中的101寝室，当别人问你住在哪个寝室时，你可以直接回答101寝室，也可以回答某省某市某区某学校101寝室。二者的区别是，前者是相对于学校而言在哪个寝室，后者则是相对于上帝视角而言在哪个寝室。

相对路径，顾名思义，相对于当前工作目录的路径。它的起点是当前工作目录而非根目录；

绝对路径，从文件系统的根目录开始的完整路径。它的起点根目录，直接指明文件的位置。

如果用相对路径来访问其他文件，那么可以试一试../，它表示的是上一级的目录（./则是表示当前目录，可要可不要），看下图：

我们先围绕一下linux的常见命令来说，因为在使用linux时我们绝大多数的时间还是要跟终端打交道的，命令是必须要学滴。

（终端，即Terminal，是用于与操作系统进行交互的接口，通过在终端输入相关命令来运行程序，查看输出，与操作系统交互等）

linux中常用的几个命令

按功能分类来说比较好一点，这样就可以知道在某些命令被过滤的情况下有哪些命令可以替代使用

文件目录和操作

  1. ls：展示某个目录下的所有子目录以及文件（隐藏文件默认不展示），如果后面没有跟路径名，则默认展示当前目录；
  2. cd：进入某个目录下，后面需要跟路径名；
  3. pwd：输出当前目录的绝对路径；
  4. mkdir/rmdir：创建一个目录；
  5. cp：将某个文件或目录复制到另外一个文件或目录；
  6. mv：移动或重命名某个文件或目录；
  7. find：查找文件或目录，输出其地址；
  8. tree：显示目录的分支结构；
  9. TO BE ADDED

文件内容和查看

  1. cat/tac：前者按行号正序输出文件内容，后者按行号倒序输出文件内容；
  2. more/less：分页显示文件内容；
  3. head/tail：默认情况下前者显示文件的前十行，后者显示文件的后十行；
  4. nl：显示文件内容并标明行号；
  5. touch：创建文件；
  6. rm：和rmdir差不多，不过这里针对的是文件；
  7. vi/vim：编辑文件；
  8. TO BE ADDED

权限管理

  1. chmod：更改文件和目录的权限
  2. chown：更改文件和目录的拥有者
  3. passwd：更改某用户的密码
  4. sudo：以管理员身份执行某条命令
  5. TO BE ADDED

压缩与打包

  1. tar：打包某个文件或目录（有的博客会说它有压缩和解压功能，实际上它本身并不具有这些功能，只能通过调用其他其他的功能来完成这一过程）

  2. gzip：将打包后的文件或目录进行压缩（一般后缀是tar.gz，因为它一般都是压缩tar文件）

  3. zip/unzip：压缩或解压缩zip文件

  4. TO BE ADDED

网络管理

  1. wget/curl：从某个网站中获取文件或其他信息
  2. netstat：显示网络连接信息
  3. nc：连接目标主机的终端
  4. ping：检测本地与目标的网络是否通畅
  5. ssh：本地远程登录目标主机
  6. TO BE ADDED

进程管理

  1. ps：查看进程信息，ps aux则是查看所有进程的信息
  2. kill：停止进程的运行
  3. TO BE ADDED

其他

  1. whoami：查询自己是哪一位用户
  2. echo：显示指定的文字，一般与重定向使用
  3. TO BE ADDED

一般情况下，命令是有一个-h，–help或help选项，当你不知道或忘了某个选项时可以输入他们来看看哪些选项可用，如果英语看不惯，那就上网多查查吧。

额外哆嗦的一些命令（附加，视情况食用）

文件内容和查看

  1. sort：给文件的每一行进行排序，默认是字典排序
  2. uniq：将文件内容进行去重后输出
  3. od：把文件转换成某种形式输出，包括字符，ascii，二进制，八进制，十六进制

:warning:上面几个命令不会直接对文件内容作任何更改的，只是输出的内容可能跟文件内容不一样

  1. strings：查找文件中指定内容并输出，默认情况下是查找长度至少为4的字符串
  2. cut：按照输入的条件选取文本内容进行输出
  3. paste：将指定的文件以列为单位进行合并并输出
